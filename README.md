Task Manager:
A simple Task Manager application with JWT authentication and role-based access control.
Built with Node.js, Express, MySQL/Sequelize, and a simple React frontend.

Features:
1.User authentication with JWT
2.Role-based access:
Admin can see all tasks.
User can manage only their own tasks
3.CRUD operations for tasks: Create, Read, Update, Delete
4.Simple React frontend for interaction
5.API documentation--> (Postman) 

Tech Stack:
1.Backend: Node.js, Express, Sequelize, MySQL
2.Frontend: React (Vite)
3.Authentication: JWT
4.Password Hashing: bcryptjs
5.Validation: express-validator
